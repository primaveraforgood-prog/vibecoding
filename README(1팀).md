# 260526 바이브 코딩 발표자료 사용법

## 1. 자료 개요

이 자료는 Times of India의 「What is 'vibe coding'? Former Tesla AI director Andrej Karpathy defines a new era in AI-driven development」 기사를 바탕으로 제작한 HTML 기반 발표자료입니다.  
주제는 Andrej Karpathy가 설명한 바이브 코딩의 개념, 작업 흐름, 장점, 한계, 활용 방식입니다.

- 팀: 1팀
- 작성자: 이다혜(경영 20, 1팀)
- 발표자료 형식: HTML 프레젠테이션
- 슬라이드 수: 총 15장
- 배포용 파일: PDF

## 2. 구성 파일

압축 파일에는 다음 파일이 포함되어 있습니다.

1. `바이브_코딩_1팀_양준모.html`  
   발표용 HTML 파일입니다. 웹 브라우저에서 실행하면 슬라이드 쇼 형식으로 발표할 수 있습니다.

2. `발표(양준모).pdf`  
   발표 때 배포할 수 있는 PDF 파일입니다. HTML의 움직이는 배경은 PDF에서는 정지된 화면으로 반영했습니다.

3. `README(1팀).md`  
   발표자료 사용법과 조작 방법을 설명하는 파일입니다.

## 3. 실행 방법

1. 압축 파일을 해제합니다.
2. `바이브_코딩_1팀_양준모.html` 파일을 더블클릭합니다.
3. 기본 웹 브라우저가 열리면 발표를 시작합니다.
4. 전체 화면 발표가 필요하면 `F` 키를 누르거나 우측 상단의 `전체화면` 버튼을 클릭합니다.

이 HTML 파일은 외부 인터넷 연결 없이 실행되도록 CSS와 JavaScript를 파일 내부에 포함했습니다. 따라서 Windows와 macOS 모두에서 오프라인 실행이 가능합니다.

## 4. 조작 방법

| 기능 | 키보드 / 마우스 조작 |
|---|---|
| 다음 슬라이드 | 오른쪽 방향키 `→`, 스페이스바 |
| 이전 슬라이드 | 왼쪽 방향키 `←` |
| 첫 슬라이드로 이동 | `Home` |
| 마지막 슬라이드로 이동 | `End` |
| 전체화면 전환 | `F` |
| 인쇄 또는 PDF 저장 | `P` |
| 도움말 열기/닫기 | `H` |
| 도움말 닫기 | `Esc` |
| 마우스 클릭 | 화면 오른쪽 절반 클릭: 다음 / 화면 왼쪽 절반 클릭: 이전 |

발표 중 조작 방법이 기억나지 않으면 `H` 키를 누르면 도움말 창이 표시됩니다.

## 5. PDF 사용 방법

`발표(양준모).pdf` 파일은 발표자료를 나눠주거나 제출할 때 사용할 수 있는 정지형 PDF입니다.  
HTML 발표자료의 움직이는 배경 효과는 PDF에서는 움직이지 않으며, 각 슬라이드가 한 페이지씩 정리되어 있습니다.

## 6. 주의사항

- HTML 파일명이나 함께 들어 있는 PDF, README 파일은 압축 파일 안에서 그대로 유지하는 것이 좋습니다.
- 발표 전에는 반드시 학교 PC 또는 발표용 PC에서 HTML 파일을 한 번 실행해보는 것이 좋습니다.
- 브라우저 전체 화면 모드에서 가장 보기 좋게 구성되어 있습니다.
- 개인정보가 남지 않도록 사용한 PC에 파일을 저장해두지 않거나, 발표 후 삭제하는 것을 권장합니다.
